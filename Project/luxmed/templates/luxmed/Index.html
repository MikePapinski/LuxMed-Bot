About LuxMed Bot

This is simple Web App that connects with LuxMed.pl portal and moniotors visits for you. Avaliable features are:

  • Searching for New Visits every 1 minute
  • Send instant notification via WhatsApp when new visit appears
  • Look for visits with predifened time range by the user
  • Notification when found visit disapears

How does that work?

The application is based on Python Django framework, Celery task schedules, Postgress database and all of this is dockerized in to standalone containers.
